7.9笔记

Linux系统命令的使用
mkdir make directory 创建目录 mkdir DIRNAME
ls list 显示目录中的文件 ls
touch touch 创建文件 rm FILENAME
cd change directory 切换目录 cd DIRNAME
rm remove 删除目录 rm FILENAME
命令结构
命令 【-选项】 【参数】
多个选项
1.可以写在一起
2.不区分选项的顺序

yum list 显示软件列表
yum install APPNAME 安装指定软件
yum search APPNAME 搜索相关软件
yum check-update 检查是否需要更新
yum update APPNAME 更新指定软件
yum update 更新所有软件
yum remove APPNAME 卸载指定软件
